# Student Report Generator 🎓

A Python script that analyses student data and generates a summary report.

## What it does
- Shows total number of students
- Shows how many passed and failed
- Calculates average, highest and lowest marks
- Displays failed students list
- Displays top scorer

## Tech Used
- Python
- Pandas

## How to Run

Install pandas first:
```bash
pip install pandas
```

Then run:
```bash
python student_report.py
```
